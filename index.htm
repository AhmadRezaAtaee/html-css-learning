What is HTML?

A markup language that is at the core of web page building technology alongside CSS.

The HTML Consortium develops HTML as the standard for marking up hyperlink documents for web marketing and is responsible for maintaining and developing it.

A major application and a practical example.

With a browser you request to download a web page from the server. This page is either a static source or its information is provided dynamically by the server. Either way, the feedback from the web server is done with HTML markup.

The information has reached you. You can see its structure and show it in any format, in the style you like. This is what your browser does. Each browser can display HTML tagged information in its own style.

You can also display the information marked in this language with the style you want. The solution is CSS. So wait a minute.

Different versions of HTML

HTML history has several versions, which can be called versions 2, 3.2, 4 and 5.. You can read the full HTML history. But the important thing is that different versions have tried to make this language more accurate for marking information
